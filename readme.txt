A simple plugin to tests if your password is strong or not.

== Description ==
KS Password tester help you to check/test the password you want to use is strong enough or not.

It also helps you to generate better secured password.

== Installation ==
1. Upload the `ks-password-tester` folder to the `/wp-content/plugins/` directory
2. Activate the KS Password Tester
